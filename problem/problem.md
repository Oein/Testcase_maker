# 설명

상호는 코고에서 N 찍기에 큰 재미를 느껴 N 찍기 2를 만들고 싶어한다.

상호를 도와 프로그램을 작성하자!

---

# 입력

첫째 줄에 N(1 ≤ N ≤ 50)이 주어진다.

# 출력

첫째 줄부터 N번째 줄까지 차례대로 숫자들을 출력한다.
단 N을 그대로 출력하면 출력이 이쁘지 않다고 상호가 찡찡되니 N이 10이상이면 마지막자리를 출력하자.
